conectar()
Establece la conexión con la base de datos PostgreSQL usando los parámetros de configuración
definidos en el archivo config.py. Devuelve la conexión y el cursor para ejecutar consultas.
 
Retorna:
- conn: objeto de conexión a la base de datos.
- cursor: cursor asociado a la conexión, usado para ejecutar consultas SQL.
 
Si ocurre un error al conectar, imprime el mensaje de error y retorna None, None.
